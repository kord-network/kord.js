# @meta.js/identity

> meta.js identity module

## Install
```
npm install @meta.js/identity
```

## Usage
```
// ES6
import { identity } from 'meta.js'
import * as identity from '@meta.js/identity'

// CommonJS
const { identity } = require('meta.js')
const identity = require('@meta.js/identity')
```
