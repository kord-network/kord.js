# @kord.js/identity

> kord.js identity module

## Install
```
npm install @kord.js/identity
```

## Usage
```
// ES6
import { identity } from 'kord.js'
import * as identity from '@kord.js/identity'

// CommonJS
const { identity } = require('kord.js')
const identity = require('@kord.js/identity')
```
